# isamples_webui

Web interface to iSB and iSC APIs


## Development

Generally following guidelines under https://modern-web.dev/

Setup:
```
npm install
```

Build:
```
npm run build
```
Results are in the `dist` folder.

Run local server:
```
npm run serve
```

Watch sources, build on change, and refresh view:
```
npm run watch
```
